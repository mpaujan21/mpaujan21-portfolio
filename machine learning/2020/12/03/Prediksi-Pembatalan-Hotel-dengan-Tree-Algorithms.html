Dalam project ini, saya menggunakan dataset informasi booking hotel dari Kaggle. Disini saya akan memprediksi booking hotel yang berpotensi dibatalkan dengan machine learning. Model yang akan digunakan yaitu model classification yang termasuk dalam supervised learning dan metrik evaluasi yang akan saya gunakan yaitu auc score yang akan saya jelaskan lebih lanjut.

Machine Learning

Pada machine learning ini akan digunakan tiga model klasifikasi, yaitu DecisionTree, RandomForest, dan ExtraTrees. Pada tiap model akan dilakukan GridSearchCV dahulu untuk mencari parameter mana yang memberikan skor terbaik. Berikut contoh penerapan GridSearchCV pada DecisionTree.

from sklearn.model_selection import GridSearchCV
from sklearn.tree import DecisionTreeClassifier

params_dt = {'criterion': ['gini', 'entropy'],

Selanjutnya dilakukan pemisahan data menjadi train dan test set sehingga train set bisa digunakan pada tiap model. Lalu model tersebut digunakan untuk memprediksi pada test set.

from sklearn.ensemble import RandomForestClassifier, ExtraTreesClassifier
from sklearn.model_selection import train_test_split

y = df['is_canceled']
X = df.drop('is_canceled', axis=1)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=21, stratify=y)

dt = DecisionTreeClassifier(criterion='entropy', 
                            max_features='auto', 
                            min_samples_leaf=5, 
                            min_samples_split=2)
dt.fit(X_train, y_train)
dt_pred = dt.predict(X_test)

rf = RandomForestClassifier(n_estimators=100, 
                            min_samples_leaf=5, 
                            min_samples_split=2)
rf.fit(X_train, y_train)
rf_pred = rf.predict(X_test)

et = ExtraTreesClassifier(n_estimators=100, 
                          min_samples_leaf=5, 
                          min_samples_split=2)
et.fit(X_train, y_train)
et_pred = et.predict(X_test)

Hasil Klasifikasi

Hasil dari prediksi model tersebut dapat diplot menjadi confusion matrix. Berikut penjelasannya:

  • True Positives: Memprediksi 0 dan hasilnya juga 0
  • False Positives: Memprediksi 0 tetapi hasilnya 1
  • True Negatives: Memprediksi 1 dan hasilnya juga 1
  • False Positives: Memprediksi 1 tetapi hasilnya 0

Contohnya, model Random Forest memiliki hasil True Positives sebesar 94%.

Selanjutnya dapat diplot kurva ROC untuk ketiga model, dengan penjelasan sebagai berikut:

  • False Positive Rate (FPR) = False Positive / (False Positive + True Negative)
  • True Positive Rate (TPR) = True Positive / (True Positive + False Negative)

Jadi, semakin besar AUC (Area Under Curve) semakin akurat model dalam memprediksi hasil positif maupun negatif. Disini RandomForest dan ExtraTrees memiliki AUC paling besar yaitu 0.95.

Dapat dilihat juga Feature Importance yaitu seberapa berpengaruh tiap variabel terhadap target. Disini deposit_type, country, dan lead_time adalah 3 variabel yang paling berpengaruh.
