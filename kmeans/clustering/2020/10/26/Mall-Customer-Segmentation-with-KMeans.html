The graph shows that we can segment customer into 5 clusters, but let's do that by using KMeans clustering.

KMeans has a n_clusters parameter, which is a number of cluster, that we can find the optimal n_clusters by using elbow method.

from sklearn.cluster import KMeans

df_1 = df[['Annual Income (k$)', 'Spending Score (1-100)']]
inertias = []
for i in range(1,11):
    km = KMeans(n_clusters=i, random_state=21)
    km.fit(df_1)
    inertias.append(km.inertia_)
    
plt.plot(range(1,11), inertias)
plt.title('Elbow Method using Inertia')
plt.xlabel('No of Clusters')
plt.ylabel('Inertia')

Text(0, 0.5, 'Inertia')

The inertia starts decreasing slowly at 5 n_clusters, so we'll choose 5 clusters.

km = KMeans(n_clusters=5, random_state=21)
df_1['K_Means'] = km.fit_predict(df_1)

# I ran the code without the labels below first, then I name the labels based on the scatterplot
labels = ['Low Income Low Spending', 'Low Income High Spending', 'Average ', 'High Income Low Spending', 'High Income High Spending']

plt.figure(figsize=(12,6))
sns.scatterplot(x='Annual Income (k$)', y='Spending Score (1-100)', hue='K_Means', data=df_1, palette='bright')
plt.legend(labels=labels, bbox_to_anchor=(1.05, 1), loc='upper left')
plt.title('KMeans Clustering of Annual Income vs Spending Score')
