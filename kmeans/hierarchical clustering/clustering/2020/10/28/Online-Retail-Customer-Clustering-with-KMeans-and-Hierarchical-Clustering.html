We'll use the elbow method to find optimal number of clusters.

from sklearn.cluster import KMeans

inertias = []
for i in range(1,11):
    km = KMeans(n_clusters=i, random_state=21)
    km.fit(new_df_scaled)
    inertias.append(km.inertia_)
    
plt.plot(range(1,11), inertias)
plt.title('Elbow Method using Inertia')
plt.xlabel('No of Clusters')
plt.ylabel('Inertia')

Clustering with Hierarchical Clustering 

from scipy.cluster.hierarchy import linkage
from scipy.cluster.hierarchy import dendrogram

mergings = linkage(new_df_scaled, method='average', metric='euclidean')
dendrogram(mergings)
plt.show()

We see that the red line is when number of cluster are optimal, so we'll cut the tree into 3 clusters.

from scipy.cluster.hierarchy import cut_tree

cluster_id = cut_tree(mergings, n_clusters=3).reshape(-1,)
cluster_id

array([0, 0, 0, ..., 0, 0, 0])
new_df1 = new_df.copy()
new_df1['Cluster'] = cluster_id
